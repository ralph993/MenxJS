# MenxJS

### A lightweight CLI to create APIs and APIs components in an easy way

## Directory Tree

```
 app/

 │ │ ├─ index/
 │ │ │ ├─ index.controller.js
 │ │ │ ├─ index.middleware.js
 │ │ │ ├─ index.router.js
 │ │ │ ├─ schema/
 │ │ │ │ ├─ index.schema.js

 │ ├─ routes/

 │ │ ├─ routes.json

```

```
 npm i menxjs -g

```
